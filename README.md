# COGCS

Projeto de estudo em transicao para open source.

## O que e

COGCS nasceu de uma dor real no meu processo de aprendizado.

O objetivo e explorar:

- repeticao espacada;
- aprendizado ativo;
- decisoes de produto com escopo pequeno;
- documentacao publica dos aprendizados.

## Estado atual

Este repositorio esta sendo aberto gradualmente.

Nao e um produto comercial nem um SaaS paralelo.
E um laboratorio de estudo e engenharia para validar problema, escopo e evolucao do projeto em publico.
